## Instalação

Estas skills seguem o padrão aberto [Skills.sh](https://skills.sh/) e podem ser instaladas em qualquer agente compatível (Claude Code, Cursor, Codex, Gemini CLI, GitHub Copilot, Windsurf e [outros](https://skills.sh/)).

### Instalar todas as skills

```bash
npx skills add fabioassuncao/agentskills --all
```

### Instalar uma skill específica

```bash
npx skills add fabioassuncao/agentskills --skill digital-product-analysis
```

### Instalar globalmente (disponível em todos os projetos)

```bash
npx skills add fabioassuncao/agentskills --skill digital-product-analysis -g
```

### Instalar via URL

```bash
npx skills add https://github.com/fabioassuncao/agentskills
```
